Get xorg to run su/sudo apps {#Get xorg to run su/sudo apps}

The default for Ubuntu (and others?) is that it's not allowed to run xorg apps from su/sudo. This was the case with the "Linux apps" in the Acer chromebook. They are allowed with a line in the .profile file (from Arch https://wiki.archlinux.org/index.php/Running_GUI_applications_as_root)

```

Add the following line to /etc/profile:

export XAUTHORITY=/home/username/.Xauthority ```
